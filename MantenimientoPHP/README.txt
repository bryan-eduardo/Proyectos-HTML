El principal uso de esta pagina es enfocado para verificar si ciertas empresas cuentan con vacantes o no cuentan con vacantes
Especialmente esta enfocado a la parte del administrador para colocar las empresas pidiendo los siguientes datos:

- Nombre de la empresa
- Codigo postal
- Puesto en la empresa
- Puesto en la empresa
- Razon social
- Telefono
- Calle y numero
- Colonia o alcaldia

- Codigo

El codigo es una forma de identificar y facilitar el borrado en la lista que aparecera con su previa coneccion a una base de datos (en este caso se uso SQL SERVER)


Tambien cuenta con botones para modificar, editar y eliminar datos que se ingresen
